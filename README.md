# calc-rpc
Calculadora das quatro operações via RPC

para construir:
```bash    
    $ make
```

para executar no servidor
>é necessário estar executando o rpcbind
```bash    
    $ ./calc_server
```    
para executar no cliente
```bash    
    $ ./calc_client
```
depois é só informar os 2 numeros e a operação.
